# Nassau Candy Decision Intelligence Dashboard

Streamlit-based dashboard for lead time prediction, shipping analytics, and factory scenario simulation.

## Files
- `app.py`
- `train_model.py`
- `simulation_engine.py`
- `recommendation_engine.py`
- `factory_mapping.py`

## Run locally
```bash
pip install -r requirements.txt
python train_model.py
streamlit run app.py
```
